# dotfiles

- OS: [Arch](https://archlinux.org/)
- Compositor: [Wayfire](https://github.com/WayfireWM/wayfire)
- Status bar: [Ergo](https://github.com/d4yr41n/ergo)
- Fonts: [Fira Code](https://github.com/tonsky/FiraCode), [Nerd Fonts Symbols](https://github.com/ryanoasis/nerd-fonts/tree/master/patched-fonts/NerdFontsSymbolsOnly)
- Menu: [Wmenu](https://sr.ht/~adnano/wmenu)
- Terminal: [Alacritty](https://github.com/alacritty/alacritty)
- Text editor: [Helix](https://github.com/helix-editor/helix)
- Browser: [Chromium](https://github.com/chromium/chromium)
- Shell: [Bash](https://git.savannah.gnu.org/cgit/bash.git)
